## 12주차 과제: 세트 (Set)

`파이썬실습_#6.pdf` (p.31, 32)에 포함된 **세트(Set) 활용 과제 #32~#38** 제출입니다.
교재의 개념과 메서드 활용법을 참고하여 작성하였습니다.

---

### 과제 목록 및 주요 학습 내용

**과제 #32 (집합 연산: 합집합, 교집합)**
- `|` 연산자와 `&` 연산자를 사용하여 두 세트의 합집합과 교집합을 구했습니다.
- (참고: 교재 p.8~9)

**과제 #33 (집합 연산: 차집합, 대칭차집합)**
- `-` 연산자(차집합)와 `^` 연산자(대칭차집합)를 활용했습니다.
- (참고: 교재 p.10~11)

**과제 #34 (값 할당: update)**
- `update()` 메서드를 사용하여 기존 세트에 새로운 값을 추가했습니다.
- `update`에는 리스트나 세트 형태의 iterable 객체를 전달해야 함을 확인했습니다.
- (참고: 교재 p.12)

**과제 #35 (집합 갱신: _update 메서드)**
- `intersection_update`: 교집합으로 갱신
- `difference_update`: 차집합으로 갱신
- `symmetric_difference_update`: 대칭차집합으로 갱신
- 각 메서드가 원본 세트(`a`)를 직접 변경한다는 점을 확인하고 구현했습니다.
- (참고: 교재 p.13~15)

**과제 #36 (부분집합과 상위집합)**
- `issuperset()`: 상위집합 여부 확인 (`>=` 개념)
- `issubset()`: 부분집합 여부 확인 (`<=` 개념)
- 두 조건이 모두 참일 경우 "동시"를 출력하도록 조건문을 작성했습니다.
- (참고: 교재 p.16, p.18)

**과제 #37 (요소 추가 및 삭제)**
- `add()`: 요소 1개 추가
- `discard()`: 특정 값을 지정하여 삭제 
- (참고: 교재 p.22~23)

**과제 #38 (Set Comprehension)**
- `if` 조건문이 포함된 세트 표현식을 사용하여 3과 5의 공배수를 추출했습니다.
- (참고: 교재 p.30)
